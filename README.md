# TaskFlow

A full-stack task management app with authentication, built with React, Node.js, Express, and PostgreSQL.

## Tech Stack

**Frontend:** React, Vite, React Router, Context API, Tailwind CSS
**Backend:** Node.js, Express
**Database:** PostgreSQL
**Auth:** Supabase Auth
**Deployment:** Vercel (frontend), Render/Railway (backend)

## Features

- User authentication (sign up, login, logout, protected routes)
- Create, edit, delete, and complete tasks
- Filter tasks (all, active, completed, high priority)
- Search tasks
- Sort by due date, priority, or alphabetically
- Task completion statistics
- Fully responsive design
